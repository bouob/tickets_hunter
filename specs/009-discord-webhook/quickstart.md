# Quickstart: Discord Webhook 通知

**分支**：`009-discord-webhook` | **日期**：2025-12-04

---

## 快速開始

### 步驟 1：建立 Discord Webhook

1. 開啟 Discord，進入你的伺服器
2. 右鍵點擊目標頻道 → **編輯頻道**
3. 選擇 **整合** → **Webhooks** → **新 Webhook**
4. 命名 Webhook（如「搶票通知」）
5. 點擊 **複製 Webhook URL**

### 步驟 2：設定 Tickets Hunter

1. 開啟 Tickets Hunter 設定介面 (`http://127.0.0.1:16888/`)
2. 切換到 **進階設定** 頁籤
3. 找到 **Discord Webhook URL** 欄位
4. 貼上剛才複製的 Webhook URL
5. 點擊 **儲存**

### 步驟 3：測試通知

執行搶票程式，當系統找到票券或訂單成功時，你會在 Discord 頻道收到通知。

---

## 通知觸發時機

| 時機 | 通知內容 | 說明 |
|------|----------|------|
| **找到票券** | `[平台] 找到票券！請儘速查看電腦` | 系統發現可購買的票券 |
| **訂單成功** | `[平台] 訂單成功！請儘速結帳付款` | 進入結帳/確認頁面 |

---

## 停用通知

將 **Discord Webhook URL** 欄位清空並儲存即可停用。

---

## 常見問題

**Q: 沒收到通知怎麼辦？**
- 確認 Webhook URL 正確
- 確認 Discord 頻道通知設定已開啟
- 檢查是否已觸發搶票成功事件

**Q: 會不會影響搶票速度？**
- 不會。通知使用背景執行緒發送，不影響主流程。

**Q: 通知失敗會怎樣？**
- 靜默處理，不影響搶票。可開啟 verbose 模式查看錯誤。
